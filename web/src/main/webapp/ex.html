태그(Tag)는 문서의 구조와 형태를 표현하는 명령어로 < >사이에 적어준다
출력 방식에 따른 두가지 형태를 가진다

1) Block 방식 : 새로운 행에 출력(자동 라인 스킵) p, h, div, strong ...
2) Inline 방식 : 같은 행에 출력 a, b, i, span ...
